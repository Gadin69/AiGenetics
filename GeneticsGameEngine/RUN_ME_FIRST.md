# 🚀 Run Your 3D Genetics Game Engine

## Quick Start Instructions

To see your 3D game environment immediately, follow these steps:

### Option 1: PowerShell (Recommended)
1. Open PowerShell in the `GeneticsGameEngine` folder
2. Run this command:
```powershell
.uild.ps1
```

### Option 2: Command Prompt
1. Open Command Prompt in the `GeneticsGameEngine` folder
2. Run this command:
```cmd
build.bat
```

### Option 3: Manual Build
1. Open Visual Studio 2022
2. File → Open → Solution → Select `GeneticsGameEngine.sln`
3. Build → Build Solution
4. Press Ctrl+F5 to run

## What You'll See
- A 1920×1080 window titled "3D Genetics Game"
- Real-time FPS counter printing to console
- The foundation for your genetics-driven 3D creatures

## Requirements
- Visual Studio 2022 with C++ development tools
- Windows 10/11
- DirectX 12 compatible GPU
